ICSAC Editorial System

Open-source AI editorial pipeline for the Institute for Complexity Science and Advanced Computing.

Source (Files) README Rubrics icsacinstitute.org /submit

This Space hosts the canonical source of the system that receives, reviews, and adjudicates every paper submitted to ICSAC and its journal Persistence. The submission handler, the rubrics, the prompts, the citation checks, the redaction layer, the audit pass that scores the panel itself — nothing about how submissions are evaluated is hidden, with one defensive exception (the prompt-injection signal in the RQC audit is unpublished).

Why this is public

Independent and heterodox researchers have legitimate cause to be skeptical of black-box editorial AI. The reasonable response is not to hide the system behind a "trust us"; the reasonable response is to publish it. Open the README for the full overview, or browse the source via the Files tab above.
